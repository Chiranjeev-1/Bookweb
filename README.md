# Bibilocheck




## Introduction:

Hey Everyone!! This project is a Web Application for all the Book Readers and Creative Writers.

You can Store Information about the Books you have read or planning basically a database for all necessary information about your Books.

If you have a creative side, and pass your time writing either professionally or not you can upload your work here and ask for review if you want or even just store it.

You can also discuss Stuff on Forum and follow the people that catch your interest.
